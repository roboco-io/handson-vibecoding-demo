## 참고사항

- 코어 비즈니스 로직(Todo 모델, 상태 관리, 스토리지 어댑터, 커스텀 훅)은 반드시 TDD 방식으로 구현합니다.
- 각 작업이 완료될 때마다 해당 체크리스트 항목을 체크하고 커밋합니다.
- UI 컴포넌트는 Mantine을 사용하여 구현합니다.
- 모든 코드는 TypeScript로 작성하며, 적절한 타입 정의를 포함해야 합니다.
- 접근성(WCAG 2.1 AA 수준)을 고려하여 구현합니다.
